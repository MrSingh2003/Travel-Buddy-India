# **App Name**: Travel Buddy India

## Core Features:

- Personalized Itinerary Generation: AI-powered tool analyzes user preferences, travel dates, and budget to generate optimized travel itineraries.
- Local Cab Booking with Verification: Provides a list of local cab services in areas without Ola or Uber, including verified ID documents and contact details.
- Offline Route Navigation: Allows users to download maps and routes for offline access in areas with limited network connectivity.
- AI Travel Assistant Chatbot: Offers real-time support and answers travel-related questions via an intelligent chatbot.
- Transparent Pricing Display: Clearly displays pricing for all services, including transportation, accommodations, and activities, ensuring no hidden fees.
- Integrated Hotel and Dharamshala Information: Provides comprehensive information on hotels and dharamshalas, including reviews, amenities, and booking options.
- Travel Vehicle Booking: Enables booking of travel vehicles for city and state travel, like buses and trains.
- User Authentication: Allows user authentication using their mobile number.

## Style Guidelines:

- Primary color: Vibrant orange (#FF8C00) to evoke a sense of adventure and warmth.
- Background color: Light, desaturated orange (#FFF2E0) to provide a clean and inviting backdrop.
- Accent color: Yellow-orange (#FFA500) to highlight important information and call-to-action buttons.
- Body and headline font: 'PT Sans' for a modern and readable interface.
- Use clean, recognizable icons for navigation and key features.
- Emphasize a clean and intuitive design with easy navigation.
- Subtle transitions and animations to enhance user experience and provide feedback.
